# NEXUS MIND

**NEXUS MIND is a PoW-anchored AI trading agent identity NFT collection, formed through HACD Stack.**

Each identity is backed by one HACD, with real PoW formation cost and a permanent on-chain identity/archetype.

---

## Overview

| Parameter | Value |
|-----------|-------|
| Asset type | NFT |
| Total supply | 256 |
| HACD lots | 256 (1 NFT per lot) |
| Archetypes | 4 (Momentum, Contrarian, Arbitrageur, Sentinel) |
| Phase model | Single public phase |
| Max per wallet | 4 HACD |
| Removal effect | Burn |
| Platform | HACD Stack / HACD Launchpad |

---

## Archetypes

Archetype is determined by the first letter of the HACD inscription:

| Archetype | Inscription Letters | Count | Bias |
|-----------|-------------------|-------|------|
| Momentum | W, T, Y, U | 80 | Bullish |
| Contrarian | I, A, H, X | 64 | Inverse |
| Arbitrageur | V, M, E, K | 64 | Neutral |
| Sentinel | B, S, Z, N | 48 | Risk |
